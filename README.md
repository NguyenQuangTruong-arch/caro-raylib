A minimalist, liminal-space inspired Gomoku (Caro) game written in C using the Raylib library.


## Features
* **Game Modes**: PvP or PvE
* **AI**: simple greedy algorithm that evaluates the next best possible move.
* **Save & Load System**: Includes 3 persistent save slots, reading and writing game states via binary `.dat` files.
* **Control**: Navigable using Mouse or Keyboard (WASD/Arrows to move, Space/Enter to select).
* **Audio**: Background music and UI sound effects with volume slider and toggle switch.

## This project is made using
* **Language**: C
* **Library**: [Raylib](https://www.raylib.com/) (Graphics, Audio, Input handling)
* **Environment**: Microsoft Visual Studio

## Getting Started

### Prerequisites
* Microsoft Visual Studio (with C++ Desktop Development workload installed).

### Installation & Running
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/NguyenQuangTruong-arch/caro-raylib

2. Navigate to the project directory and open the caroGame.sln file using Visual Studio.

3. Ensure the Solution Platform is set to x64.

4. Press F5 (or click Local Windows Debugger) to build and run the game.
